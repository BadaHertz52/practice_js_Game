#토끼 게임

##게임 소개
해당 게임은 토끼를 점프시켜 점수를 얻고 레벨업을 하여 미션을 완수하는 게임으로 HTML,CSS, Java Script를 사용하여 만들었습니다.
해당 게임을 진행하기 위해서 게임 특성상 768px이상의 화면 크기가 필요하며 화면이 768px 미만인 경우 이에 대한 알림창이 뜨도록 설정하였습니다.
